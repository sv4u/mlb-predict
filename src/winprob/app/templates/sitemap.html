⚡ Data API

GET/api/version
Application version and git commit hash.
GET/api/seasons
List of all available seasons with data.
GET/api/teams
All teams with Retrosheet codes and full names.
GET/api/games
Query games with filters: season, home/away team, date, pagination, and sort order.
GET/api/games/{game_pk}
Full feature breakdown and SHAP attribution for a single game by game_pk.
GET/api/upsets
Biggest upsets: heavy favorites that lost. Filter by season, team, and minimum probability.
GET/api/cv-summary
Cross-validation results from the latest model training run.
GET/api/standings
Predicted vs actual standings by division with league leaders. Supports season parameter.
GET/api/team-stats
Team batting and pitching statistics for a given season from the MLB Stats API.

🛠 Admin API

GET/api/admin/status
Full system status: data coverage, model inventory, and pipeline states.
POST/api/admin/ingest
Trigger full re-ingestion of all seasons. Clears and rebuilds all processed data.
POST/api/admin/update
Update current season data only (non-destructive incremental update).
POST/api/admin/retrain
Clear all model artifacts and retrain all models from scratch.
